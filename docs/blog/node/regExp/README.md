1、特殊字符校验
---
<h5>只允许中文和破折号，例如："深圳市-母婴室"，前面后面都不能出现括折号</h5>
- reg = /^(?!-)(?!.*--)[\u4e00-\u9fa5]+-[\u4e00-\u9fa5]+(?<!-)$/；

<h5>只允许中文和破折号，例如："某某-某某-某某"，前面后面都不能出现括折号</h5>
- reg = /^(?!-)(?!.*--)[\u4e00-\u9fa5]+-[\u4e00-\u9fa5]+-[\u4e00-\u9fa5]+(?<!-)$/；

<h5>只允许英文和破折号，例如："m-m-n"，前面后面都不能出现括折号</h5>
- reg = reg = /^(?!-)(?!.*--)[A-Za-z0-9-]+(?<!-)$/；
